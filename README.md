

## Technologies

- Frontend: React
- Backend: dotnet
- Database: PostgresSql

## Getting Started

### Prerequisites
- Docker and Docker Compose (optional)

### Installation

1. Clone the repository
   ```
   git clone https://github.com/muhammadsohaib2000/Secure-Messages.git
   cd Secure-Messages
   ```

2. Install dependencies
  
   
   # Frontend
   cd ../frontend
   npm install
   ```

3. Run the application
   ```
   # Using Docker
   docker-compose up
   
   # Or run separately
   # Backend
   cd backend
  dotnet run
   
   # Frontend
   cd frontend
   npm start
   ```
